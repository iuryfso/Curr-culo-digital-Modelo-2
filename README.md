# 📄 Currículo Digital — Isabela Domingues Silva

> Currículo digital moderno e interativo desenvolvido com HTML, CSS e JavaScript puro, hospedado via GitHub Pages.

🔗 **[Acesse o site ao vivo](https://seu-usuario.github.io/curriculo-isabela)**

---

## ✨ Sobre o projeto

Este projeto é um currículo digital pessoal desenvolvido para apresentar de forma elegante e profissional a trajetória de **Isabela Domingues Silva** — profissional com experiência em atendimento ao cliente, operação de caixa, estoque, vendas e análise de crédito.

O design foi desenvolvido com foco em **estética moderna**, utilizando glassmorphism, gradientes animados, micro-animações e tipografia premium para criar uma experiência visual impactante.

---

## 🚀 Funcionalidades

- 🎨 **Design dark** com paleta pink/rosa vibrante
- ✨ **Animações** — texto em shimmer, card flutuante, borda rotacionada e glow
- 📱 **Totalmente responsivo** — funciona em desktop, tablet e mobile
- 🖼️ **Foto de perfil** no logo da navbar e no card de contato
- 🧭 **Navegação suave** com scroll animado entre seções
- 📂 **Link direto para o PDF** do currículo
- 📧 **Link para contato** via e-mail
- ⚡ **Deploy automático** via GitHub Actions → GitHub Pages

## 🛠️ Tecnologias utilizadas

- **HTML5** — estrutura semântica
- **CSS3** — animações, glassmorphism, grid e flexbox
- **JavaScript** — Intersection Observer para animações de scroll
- **Google Fonts** — Inter + Playfair Display
- **GitHub Actions** — CI/CD para deploy automático
- **GitHub Pages** — hospedagem gratuita

---

## 🚢 Deploy

O projeto é publicado automaticamente no **GitHub Pages** a cada `push` na branch `main`, utilizando o workflow do GitHub Actions definido em `.github/workflows/deploy.yml`.

Para publicar manualmente, acesse a aba **Actions** no GitHub e clique em **"Run workflow"**.
